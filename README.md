# Discord App Cloning in React JS
[![Awesome](https://awesome.re/badge-flat2.svg)](https://github.com/radespratama/discord-clone-mini-project.git#readme)

## Getting Started
#### Instalation :
<span>If you want to add this project to your own computer, you can install it by following the method below</span>
```
git clone https://github.com/radespratama/discord-clone-mini-project.git repoName
```
```
npm install | yarn install
```
```
npm start | yarn start
```

#### Built With

* [Create React App](https://reactjs.org/docs/create-a-new-react-app.html) - The web library used
* [Redux](https://react-redux.js.org/) - State Management
* [Material UI](https://material-ui.com/) - Styling & Icon
* [Firebase](https://firebase.google.com/?hl=id) - Backend, Database, and Deploy
